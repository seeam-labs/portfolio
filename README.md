# portfolio

A Vite + React portfolio starter with Tailwind CSS.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Publish to GitHub Pages

This project is configured to deploy automatically from `main` to the `gh-pages` branch.

1. Commit and push your changes to `main`.
2. The GitHub Actions workflow will build the site and publish it.
3. The site will be available at:
   `https://seeam-labs.github.io/portfolio`

## Notes

- The app uses `lucide-react` for icons.
- Add your own image file or update the hero image source in `src/App.jsx`.
- `CV_SEEAM_RAHMAN.pdf` is referenced in the hero section and should be added to the public root if needed.
